What is that product? For who?

Amazon is an American company that needs no introduction. It wasn't so obvious at first, as they started out selling used books and are ultimately seen as the largest distributor in the world.

What problem have we solved?

Customers from all over the world, shopping for various types of purchases on Amazon, needed efficient invoicing. At first glance, it may not seem complicated. However, when it was necessary to create a document compatible with the format used in a given country, problems in handling began to appear.

What solutions were used?

We created an application in Python to extract data from invoices issued by Amazon, then the data was saved to a CSV file. Our solution had to be simple and functional, thus facilitating the invoicing process in such a large concern as Amazon.

What was created?

Application for creating corrective invoices in the WF MAG application.

What is the phenomenon of the application?

The application connects to an external database to obtain a report on pending correction invoices for a given period. These invoices contained information on returns.

What were the difficulties in creating?

Developing such an application was a challenge because Asseco's documentation was poor and contained many errors or outdated information.

What technologies were used?

Python, MS SQL
