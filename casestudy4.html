CREDIT SUISSE

We are a full service digital agency that builds immesive user experience.

ROLE: BRANDING + WEB DESIGN + APP DEVELOPMENT + CONSULTING

Scroll down

What is that brand?

The name of this company is not accidental. It can be associated with gold because Switzerland is high in economic rankings and is considered the richest country in the world. We started cooperation with a Swiss bank to create a transaction management portal.

What problem have we solved?

Our task was to create an internal web application for the bank that would facilitate the work of many traders. What is it actually about? At the time of purchasing the shares, the user sends an instruction to a specific trader, who passes it to the manager for approval.

What solutions were used?

The application performs the following steps: approving, rejecting, blocking, unlocking, purchasing shares. Frontend: React, Angular, Vanilla JS, Redux, Node.js Backend: C#
